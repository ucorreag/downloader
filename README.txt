Downloader block
for Moodle 3.X.X
Author: Ulises Correa
Description: Module of moodle to downloader files

NOTE:
To install:
1. Copy and unzip the Downloader-X.X.X.zip folder to your moodle blocks folder.
2. Visit the Notifications page (Site Administration -> Notifications)
3. Click install Downloader block plugin
4. Add a block_downloader on your front page
5. Enjoy
